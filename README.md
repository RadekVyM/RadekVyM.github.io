# My Personal Portfolio

This is the source code for my personal portfolio built with Next.js. It showcases my projects, skills, and experience in a responsive, modern interface.

> 🔗 https://radekvym.github.io

## 🧠 Why Next.js?

I chose **Next.js** for this single-page site to prioritize **developer efficiency**:

* **Reusability:** Leveraged my existing React component library for a consistent UI.
* **Maintainability:** Used a modular structure to eliminate the redundancy of a single, massive HTML file.
* **Workflow:** Minimized development time by using a familiar, modern ecosystem.

> **Summary:** While a simple HTML file would suffice, Next.js offered superior speed and code organization.

## 🛠️ Tech Stack

* [Next.js](https://nextjs.org/) (App Router)
* [React Icons](https://react-icons.github.io/react-icons/)

### 🚀 Setup

```bash
npm install
npm run dev
```
